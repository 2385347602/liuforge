## 为什么建这个站

一直以来都想拥有一个完全属于自己的数字花园，可以自由地写博客、分享工具、整理资源。趁着周末，用 **Astro + Vue 3 + Cloudflare Pages** 这套技术栈，一天时间搞定了整个站点。

## 技术选型

### Astro - 内容站点的最佳选择

选 Astro 的核心原因就一个：**默认零 JavaScript**。博客和排行榜这类内容页面，根本不需要客户端 JS，纯 HTML 既快又省资源。需要交互的地方（工具集），再局部引入 Vue 组件即可。

```typescript
// Astro 中引入 Vue 组件非常简单
// 使用 client:load 指令让组件在客户端激活
<JsonFormatter client:load />
```

### Tailwind CSS 4 - 原子化 CSS 的新高度

Tailwind v4 相比 v3 有了质的飞跃：
- **CSS-first 配置**，不再需要 `tailwind.config.js`
- **更好的性能**，基于 Rust 的解析引擎
- **自动内容检测**，无需配置 content 路径

### Cloudflare Pages - 零成本部署

全球 CDN + 免费 SSL + 自动构建部署，push 代码就完事，真正零运维。
