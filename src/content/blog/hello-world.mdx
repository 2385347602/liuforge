## 为什么建这个站

一直以来，我都想拥有一个完全属于自己的数字花园，可以自由地写博客、展示项目、整理资源。趁着周末，用 **Astro + Vue 3 + Cloudflare Pages** 这套技术栈，把站点的第一版搭了起来。

这个站点不会只是一个文章列表。它更像一个个人数字展厅：文章、项目、工具、简历、时间线和一些生活碎片都会在这里汇合。

## 技术选型

### Astro

选择 Astro 的核心原因很简单：内容站默认应该尽量少发客户端 JavaScript。博客和作品页以静态内容为主，Astro 可以让它们保持轻量；需要交互的工具页面，再局部引入 Vue 组件。

```typescript
// Astro 中可以按需挂载 Vue 组件
<JsonFormatter client:load />
```

### Tailwind CSS 4

Tailwind v4 的 CSS-first 配置很适合快速迭代视觉系统。这个站点会长期调整，所以我更看重样式表达是否足够直接、维护成本是否足够低。

### Cloudflare Pages

静态站点很适合放在 Cloudflare Pages 上：构建简单、全球 CDN、免费 SSL，对个人项目来说足够省心。
